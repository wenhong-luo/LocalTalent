# LocalTalent 二期高保真验收清单

本文是 Prompt 23 的收口验收清单。所有项都应由脚本、测试或手工 Runbook 覆盖。

## 页面结构

- [ ] 首页包含顶部工具条、搜索区、职位分类墙、登录卡、公告/招聘会 tab、快捷入口、广告位、推荐企业、热招职位和浮动二维码占位。
- [ ] 找工作包含筛选栏、名企直聘入口、职位卡片、详情入口、扫码求职 CTA、投递/收藏占位。
- [ ] 找企业包含筛选栏、认证企业卡片、企业公开主页和在线职位聚合。
- [ ] 人才服务区采用发布快照卡片、筛选、标签、分页和风险提示，不出现公共简历库能力。
- [ ] 招聘会、就业政策、职场资讯、公告、HR 工具箱、帮助中心和更多服务均有公开页面。
- [ ] 登录入口覆盖求职者/招聘者登录注册 UI，短信/微信/小程序/App 为禁用占位。

## 字段白名单与安全

- [ ] 人才服务区仅展示 `snapshot_id/display_name_masked/city_code/category_code/skills_summary/experience_years/updated_at`。
- [ ] 公开职位不展示企业内部联系方式、审核材料、营业执照附件。
- [ ] 企业公开页不展示联系人手机号/邮箱、后台备注、审核材料、营业执照附件。
- [ ] 招聘会不展示报名名单、参会个人明细、签到证据或联系方式。
- [ ] 内容频道不展示候选人原始数据、后台备注或审核材料。
- [ ] 撤回下线仍由 `ConsentSnapshotFlowIT` 守护。

## 一期边界回归

- [ ] IDOR/BOLA 阻断测试仍通过。
- [ ] 审计中心 trace 查询与脱敏测试仍通过。
- [ ] 导出审批、字段级权限和 MinIO 短期下载测试仍通过。
- [ ] 对接接口仍为 stub，无真实 ATS/RPA 或受控候选池同步。
- [ ] 高风险能力仍处于占位或风险池，未升级为当前需求。

## SEO 与构建

- [ ] `npm run build` 通过。
- [ ] sitemap 覆盖 `/`、`/jobs`、`/companies`、`/portal/talent-service-area`、`/job-fairs`、`/articles/policies`、`/articles/news`、`/hr-tools`、`/help`、`/more-services`。
- [ ] 私有中心、后台和风险占位能力不作为公开 SEO 主入口。
