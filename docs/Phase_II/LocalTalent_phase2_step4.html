LocalTalent Phase-2 Step 4：求职者中心与企业中心能力细化

统一硬边界提醒：人才服务区仅展示发布快照；对接接口仅 stub；不得输出原始候选人数据；同意、撤回、数据域、字段级权限必须在服务端收口。

开发前必读卡

  • 文件定位：冻结求职者中心与企业中心的信息架构、页面状态、字段边界和验收关注点。
  • 资料来源：边界文件、两份新 MD、Step1-Step3、本地 demo、一期文档。
  • 禁止越界点：禁止受控找人才、联系解锁、会员商业化、真实支付、真实短信/微信/App、真实地图/视频/直播、真实 ATS/RPA。

1. Step4 总目标

中心目标冻结内容
求职者中心让求职者管理本人招聘流程与发布状态。overview、简历、投递、面试/签到、收藏/订阅、站内通知、同意/撤回。
企业中心让企业管理本企业招聘流程。overview、企业资料与认证、职位管理、投递池、面试邀约、导出入口。

Step4 不定义最终 API schema，不新增数据库表，不新增代码实现，只冻结中心页面范围、状态口径和验收边界。

2. 求职者中心能力细化

模块能力边界
中心首页 overview简历完成度、最近投递、最近面试/签到、收藏/订阅、通知、同意/撤回与发布状态。状态完全由服务端返回。
简历编辑与预览基础资料、教育经历、工作经历、技能标签、求职意向、附件上传入口、预览。不得把简历正文、附件对象 key、证据材料带入公开门户。
投递记录投递职位、企业、时间、状态、面试入口、签到结果摘要。candidate 只能查看本人投递。
面试/签到状态待面试、已签到、已过期、已取消。签到结果不等同于同意或授权。
职位收藏收藏职位、职位当前状态、取消收藏。本人私有数据，取消操作幂等。
搜索/职位订阅保存搜索条件、订阅关键词、城市、职类、薪资、经验。只做站内通知，不接真实短信/微信/App。
站内通知投递、面试、签到、反馈、同意/撤回、系统公告。不展开外部推送。
同意/撤回与发布状态展示是否已同意、已撤回、可发布、快照可见、撤回入口。撤回后人才服务区不可见；不展示同意证据明文。

3. 企业中心能力细化

模块能力边界
企业首页 overview认证状态、在招职位、待审核职位、投递池、面试邀约、导出申请摘要。统计必须限定本企业数据域。
企业资料与认证资料编辑、认证提交、驳回原因、重新提交、通过后发布职位。未认证企业不得上线职位；不公开审核材料。
职位管理职位列表、创建、编辑、提交审核、审核状态、上线状态、下线。企业不能直接把职位设为在线。
投递池投递状态、职位标题、候选人摘要、面试和签到状态。不是公共简历库；不得展示未授权联系方式或简历正文。
投递详情投递基础信息、流程状态、面试邀约、处理状态。校验本企业数据域；敏感字段读取写 field_access_log。
面试邀约基于本企业投递创建面试，展示时间、地点、状态、签到状态。不能对非本企业投递创建面试；签到码不得无限重放。
导出入口申请、审批状态、生成状态、下载状态。继承 export_apply 审批、字段级权限、MinIO 短期下载与审计链。

4. 双中心状态机

所有状态必须由服务端返回。前端只做视觉表达和下一步引导，不允许从字段缺失、按钮隐藏或本地缓存推断核心业务状态。

5. 字段与权限边界

  • 求职者中心：candidate 只能访问本人简历、投递、收藏、订阅、通知，不得把本人原始候选人数据带入公开门户或人才服务区。
  • 企业中心：company 只能访问本企业资料、职位、投递、面试、导出申请。
  • 公开与私有隔离：私有中心可读取必要流程摘要，但不得进入公开 SEO 页面、公开缓存、公开搜索索引。
  • 导出入口必须继承审批与字段级权限，不得绕开一期导出链。

6. 审计与验收关注点

  • candidate 只能读取本人中心数据；company 只能读取本企业中心数据。
  • 简历编辑、投递状态、收藏、订阅、通知均不进入公开门户。
  • 同意/撤回状态由服务端返回，撤回后人才服务区不可见。
  • 企业未认证通过不得上线职位，企业不能跨企业查看投递池、面试邀约或导出申请。
  • 投递池不返回未授权手机号、邮箱、简历正文、附件对象 key、证据或原始 JSON。
  • 高风险写动作写 audit_log，敏感字段读取写 field_access_log。

7. Step4 结论

Step4 冻结了二期求职者中心与企业中心的能力边界。双中心都是私有域，不是公开门户，也不是公共简历库；状态必须服务端返回，权限必须服务端强制，敏感字段必须可审计。

高保真门户优先补充：双中心能力仍是二期重要目标，但不应早于公共门户 Shell、首页首屏、找工作、找企业、人才服务区和内容频道的高保真落地。

双中心页面不承担 SEO 公开流量，不进入 sitemap，默认 noindex。
