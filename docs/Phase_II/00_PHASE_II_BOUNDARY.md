# LocalTalent Phase-2 Master Boundary

## 开发前必读卡

- **文件定位**：二期研发的主边界文件；后续 Step、蓝图、提示词、Prompt 开发和验收都必须先读本文件。
- **实现约束**：二期规划阶段只做文档；进入开发阶段后，每轮也只能执行当前 Prompt，不得跨轮偷跑；不得修改一期 Prompt 0-14 既定事实。
- **禁止越界点**：禁止公共简历库；禁止真实 ATS/RPA/受控候选池同步；禁止公开联系方式；禁止真实支付、短信、微信、小程序、App、地图、视频、直播接入。
- **下游影响**：约束 Phase II Step、demo inventory、二期蓝图、二期提示词和后续代码实现。
- **自动化验收关注点**：人才服务区字段黑名单、撤回下线、RBAC、数据域、字段级权限、审计、IDOR/BOLA、导出审批、对接接口 stub。

**统一硬边界提醒**：人才服务区仅展示发布快照；对接接口仅 stub；不得输出原始候选人数据；同意、撤回、数据域、字段级权限必须在服务端收口。

## 4. 二期规划红线

必须继承的一期红线：

- 人才服务区仅展示发布快照，不展示原始候选人数据。
- 原始候选人数据与发布快照必须分开管理。
- 对接接口仅 stub，不做 ATS/RPA 或受控候选池真实同步。
- 权限必须服务端强制：RBAC + 数据域 + 字段级权限。
- 高风险动作必须留痕：同意、撤回、导出、审核、字段访问。
- 任何新增迁移只能 forward-only，不得反复修改旧迁移。

二期规划阶段额外红线：

- 不新增代码。
- 不新增 Flyway 迁移。
- 不新增接口 schema。
- 不调整生产依赖。
- 不实现真实支付、短信、微信、小程序、App、地图服务、视频服务、直播服务。
- 高风险能力先进入风险池，不自动进入当前开发。

## 5. 高风险能力风险池

以下能力不是永远不做，而是必须先写规避方案、完成合规与安全评审，再决定是否进入后续 Prompt：

| 高风险能力 | 当前结论 | 未来评估重点 |
| --- | --- | --- |
| 公共简历库 | 禁止 | 只能改写为发布快照公开 + 企业受控检索。 |
| 受控找人才 | 暂缓 | 企业认证、候选人同意、隐私开关、字段级权限、联系解锁、审计。 |
| 联系方式解锁 | 暂缓 | 授权、权益、审批、访问次数、撤回联动、字段访问日志。 |
| 会员商业化 | 暂缓 | 权益模型、支付合规、退款、发票、审计。 |
| 真实支付 | 暂缓 | 支付供应商、资金流、风控、财务对账。 |
| 真实短信/微信/小程序/App | 暂缓 | 模板合规、退订、频控、隐私协议、供应商合同。 |
| 真实地图/视频/直播 | 暂缓 | SDK 合规、成本、内容审核、版权和隐私。 |
| 真实 ATS/RPA/受控候选池同步 | 暂缓 | 数据来源合法性、最小字段、同步责任、撤回删除联动。 |
| AI/模型分析候选人原始数据 | 暂缓 | 字段裁剪、脱敏、最小必要、供应商审查。 |

## 7. 公开门户允许与禁止

公开门户允许展示：

- 职位公开信息。
- 企业公开信息。
- 招聘会/活动信息。
- 就业政策/资讯/HR 工具箱内容。
- 发布快照白名单摘要。

公开门户禁止展示：

- 手机号。
- 邮箱。
- 简历正文。
- 附件对象 key。
- 证据材料。
- 聊天记录。
- 审核材料。
- 营业执照附件。
- 字段级权限未授权的数据。
- 已撤回候选人的发布快照或受控摘要。

## 8. 简历库/找人才边界

74cms demo 中存在“简历库”“搜索简历”“找人才”等入口。LocalTalent 二期必须改写为以下口径：

- 游客公开侧不得进入公共简历库。
- 游客公开侧只能查看人才服务区中的发布快照摘要。
- 企业侧未来可以规划“受控找人才”，但必须登录、企业认证通过、具备角色权限和数据域权限。
- 联系方式默认不可见。
- 手机号、邮箱、简历正文、附件对象 key、证据、聊天记录、审核材料不得公开输出。
- 后续如果做联系解锁，必须经过授权、权益或审批，并写 `audit_log` 与 `field_access_log`。
- 候选人撤回同意后，人才服务区和受控找人才都必须不可见。

## 9. 外部能力边界

二期规划阶段不接外部能力。后续即便规划外部能力，也必须按以下默认：

- ATS/RPA：只保留对接接口 stub，不做真实同步。
- 受控候选池：不接原始主库，不落真实连接器。
- 支付：只允许支付 stub 或权益占位，不接真实支付清结算。
- 短信/微信/小程序/App：只允许文档占位或 stub，不真实发送。
- 地图找工作：可规划页面和字段，不接真实地图 SDK。
- 视频/直播招聘：可规划入口，不接真实视频或直播服务。
