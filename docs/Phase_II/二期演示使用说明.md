# LocalTalent 二期演示使用说明

本文面向第一次接手 LocalTalent 的新同事。照着做，你可以在本地看到二期高保真门户：`首页 / 找工作 / 找企业 / 人才服务区 / 招聘会 / 就业政策 / 职场资讯 / HR 工具箱 / 登录注册入口`。

## 演示边界

- 二期演示数据只用于本地开发和验收，不是生产初始化数据。
- 演示数据通过 `scripts/seed_phase2_demo_data` 显式导入，不进入 Flyway。
- 人才服务区仍只展示 `candidate_publish_snapshot` 发布快照白名单字段，不展示手机号、邮箱、简历正文、附件、证据或原始 JSON。
- 对接接口仍为 stub，不连接 ATS/RPA、受控候选池或外部主库。
- 地图、视频、直播、短信、微信、小程序、App、支付、会员商业化均为占位或风险池，不接真实外部能力。

## 本地启动

启动 MySQL、Redis、MinIO：

```bash
docker compose -f infra/docker-compose.yml up -d
```

启动后端并让 Flyway 完成迁移：

```bash
cd backend
./mvnw spring-boot:run
```

新终端导入二期演示数据：

```bash
./scripts/seed_phase2_demo_data
```

启动前端：

```bash
cd frontend
npm ci
npm run dev
```

## 推荐演示路径

1. 访问 `http://localhost:3000/`：确认顶部工具条、搜索区、职位分类墙、登录卡、公告/招聘会 tab、广告位、扫码 CTA、推荐企业、热招职位。
2. 访问 `/jobs?city_code=310000&category_code=software`：确认找工作筛选、职位卡片、名企直聘入口、投递/收藏占位。
3. 访问 `/companies?city_code=310000&industry_code=internet`：确认认证企业列表、企业公开字段和在招职位数量。
4. 访问 `/portal/talent-service-area?city_code=310000&category_code=software`：确认只展示发布快照，不出现“简历库/搜索简历/联系解锁”。
5. 访问 `/job-fairs`、`/articles/policies`、`/articles/news`、`/hr-tools`、`/help`：确认公开内容频道可展示，报名名单和候选人明细不公开。
6. 访问 `/more-services`：确认高风险入口为禁用占位或风险池说明。
7. 访问 `/auth/login` 与 `/auth/register`：确认求职者/招聘者登录注册 UI，短信、微信、小程序、App 均为禁用占位。

## 常见问题

- `seed_phase2_demo_data` 提示核心表缺失：先启动后端，让 Flyway 迁移完成。
- 页面列表为空：确认后端仍在运行，并重新导入 `./scripts/seed_phase2_demo_data`。
- 登录失败：确认使用默认密码 `LocalTalent@123456`，并查看响应中的 `trace_id`。
- `npm run build` 后 `frontend/next-env.d.ts` 变更：这是 Next 生成噪音，只恢复该文件，不做功能改动。
