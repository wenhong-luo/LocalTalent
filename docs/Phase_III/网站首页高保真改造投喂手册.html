本轮只执行 Prompt Home-1：首页运营位模型与后台配置基础。
禁止实现 Prompt Home-1 之外的内容。

本轮只做“首页运营位/广告位”的后台可配置基础，不重做首页 UI。
禁止实现真实广告售卖、支付、点击结算、外部广告平台、短信微信、小程序/App 或联系解锁。

目标：
- 新增首页运营位配置模型，用于支持首页 banner、通栏广告、半宽广告、三列广告、快捷入口等位置。
- 运营人员可在后台创建、编辑、下线首页运营位。
- 公开首页只读取已发布、在线、未过期的运营位。
- 运营位可以配置标题、副标题、链接、排序、状态、开始/结束时间、展示位置。
- 本轮图片可先使用占位 URL 或已有静态资源路径；真实图片上传留到下一轮。

建议实现：
- 新增 forward-only 迁移，例如 V0029__create_portal_home_operation_slot.sql。
- 新增表 portal_home_operation_slot，字段至少包含 slot_code/title/subtitle/image_url/image_alt/link_type/link_url/target_type/target_id/display_order/status/start_time/end_time/operator_id/created_at/updated_at。
- 新增或扩展运营后台接口：GET/POST/PUT /api/admin/home-slots，POST /api/admin/home-slots/{id}/offline。
- 新增公开接口：GET /api/portal/home-slots?slot_codes=...
- 写操作必须鉴权 operator，auditor 只读。
- 写操作必须有 X-Trace-Id、X-Idempotency-Key 和 audit_log。
- 公开接口不得返回后台备注、operator_id、对象 key、审核材料或内部字段。

首页 slot_code 固定建议：
- home_hero_banner
- home_full_width_banner
- home_half_left
- home_half_right
- home_third_1
- home_third_2
- home_third_3
- home_quick_1
- home_quick_2
- home_quick_3
- home_bottom_banner

验收：
- operator 可以配置首页运营位。
- auditor 写操作被拒绝。
- 公开接口只返回合法在线运营位。
- 不出现真实广告支付、点击结算或外部广告平台字段。
- 后端测试、前端测试、构建和三期边界脚本通过。

本轮只做首页运营位图片资产上传和预览。

目标：
- 运营人员可为首页运营位上传、替换、删除图片。
- 图片只用于公开首页运营位展示。
- 前端和公开接口不得暴露 MinIO object key、bucket/path 或 presigned URL。

建议实现：
- 扩展 portal_home_operation_slot，如需要新增字段：image_object_key/image_file_name/image_content_type/image_size_bytes/image_sha256/image_uploaded_at。
- 新增 admin 图片接口：POST /api/admin/home-slots/{id}/image，GET /api/admin/home-slots/{id}/image/content，DELETE /api/admin/home-slots/{id}/image。
- 新增公开图片代理：GET /api/portal/home-slots/{id}/image。
- 上传限制 JPG / PNG / WebP，默认最大 5MiB。
- 校验 MIME、扩展名、非空文件、安全文件名。
- object key 由服务端生成，不使用用户原始文件名作为路径。
- 上传、替换、删除写 audit_log。
- 图片读取写必要的访问日志，但不记录 object key。

前端：
- 运营后台 home slot 表单增加图片上传、预览、替换、删除。
- 首页仍只通过公开代理图片地址展示。
- 无图时显示安全占位，不硬编码敏感或外部图片。

- 图片上传后首页可显示。
- 响应、DOM、日志不出现 object key、bucket/path、presigned URL。
- 非 operator 不可上传。
- auditor 只读。
- 三期边界脚本通过。
