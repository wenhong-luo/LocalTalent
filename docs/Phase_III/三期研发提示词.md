# LocalTalent 三期 Codex 投喂手册（可执行版 · Prompt 25–31）

> 用途：三期代码开发前必读 Markdown 版；打开对应 HTML 后可逐轮复制 Prompt 25–31。
> 重要：三期目标是“小范围真实试运营”，不是公网正式运营，也不是商业化完整版。

## 开发前必读卡

- **文件定位**：Codex 三期逐轮投喂执行手册，覆盖 Prompt 25–31，对应 P3-0 至 P3-6。
- **实现约束**：每轮只做当前 Prompt；不得跨 Prompt 偷跑；不得修改一期 Prompt 0–14、二期 Prompt 15–24 既定事实。
- **三期目标**：把一二期可演示系统推进为真实求职者、企业、运营人员可小范围灰度试运营的业务系统。
- **禁止越界点**：禁止公共简历库；禁止受控找人才、联系解锁、会员支付、真实短信、真实微信、小程序、App、真实地图、视频、直播、真实 ATS/RPA 或受控候选池同步。
- **自动化验收关注点**：人才服务区发布快照、字段黑名单、RBAC、数据域、字段级权限、IDOR/BOLA、审计 trace、导出审批、撤回下线、feature flag、灰度 Runbook。

**统一硬边界提醒**：人才服务区仅展示发布快照；原始候选人数据不得公开；对接接口仅 stub；同意、撤回、数据域、字段级权限、审计和导出审批必须由服务端收口。
