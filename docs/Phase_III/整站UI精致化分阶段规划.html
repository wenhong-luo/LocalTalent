将控件高度、字体、卡片密度、筛选区布局、按钮风格和后台工作台视觉统一到更精致的人才网后台形态。

Phase III 补充规划 Prompt UI-1 至 UI-8 只做 UI，不扩业务

Summary

整站 UI 精致化建议拆成 8 个提示词 完成。先做统一视觉规范和共享控件，再逐个业务域套用，避免把视觉优化误伤成业务改造。

  • UI-1 只改前端视觉底座和共享选择器样式，不改后端、不新增接口、不新增迁移、不改变权限或业务边界。
  • Prompt UI-2：招聘者后台 Shell 精致化 已完成。
  • UI-2 只优化 /company 顶部栏、左侧菜单、内容区宽度、背景、卡片密度和响应式，不改接口、状态机、门禁、权限或风险能力。
  • Prompt UI-3：搜索简历 UI 高保真重做 已完成。
  • UI-3 只优化 /company 搜索简历页筛选区、简历卡片、分页和空态视觉，不改搜索接口、详情/举报/申请门禁或字段黑名单。
  • 已通过前端测试、构建和三期边界脚本。

推荐 8 轮提示词

Prompt UI-1：全站视觉基线与共享控件收口

建立统一 UI tokens，重构共享选择器紧凑风格，让输入框、下拉框、按钮、标签、卡片先统一底座。

Prompt UI-2：招聘者后台 Shell 精致化

优化 /company 顶部栏、左侧菜单、内容宽度、背景和卡片密度，对齐 74CMS 后台风格。

Prompt UI-3：搜索简历 UI 高保真重做

修复搜索简历页控件不对齐、筛选行密度不统一和简历卡片不够精致的问题。

Prompt UI-4：职位管理与发布职位 UI 精致化

统一职位管理列表、发布职位长表单、回收站、表格、批量操作栏、tab 和分页样式。

Prompt UI-5：企业管理与会员首页 UI 精致化

优化企业基本资料、Logo 上传、企业风采和招聘者会员首页，不接公开展示。

Prompt UI-6：求职者端 UI 精致化

优化求职者首次完善简历、会员中心、期望职位/地区、附件上传和 AI 建议卡片。

Prompt UI-7：公开门户 UI 对齐

优化首页、找工作、找企业、人才服务区等公开页面，并修复导航高亮和广告位留白。

Prompt UI-8：全站验收、响应式和文档收口

做宽屏、笔记本、移动端回归，补测试并更新三期演示说明和领导演示路线。

推荐优先级

  • 先解决当前最扎眼的问题：做 Prompt UI-1 到 Prompt UI-3。
  • 让招聘者后台整体像专业人才网：做 Prompt UI-1 到 Prompt UI-5。
  • 整站统一交付给领导或客户看：完整做 Prompt UI-1 到 Prompt UI-8。

统一边界

  • 本规划只做 UI 精致化，不新增业务能力。
  • 不新增后端接口、数据库迁移、权限规则或真实外部能力，除非某轮单独明确。
  • 公共简历库、联系解锁、真实下载、真实支付、短信/微信/小程序/App、地图、直播、ATS/RPA 不因 UI 优化而开放。
  • 人才服务区仍只展示候选人授权发布快照。

建议下一轮

UI-3 已完成，下一轮建议执行：Prompt UI-4：职位管理与发布职位 UI 精致化。
